# Подключение аккаунта MAX в 1С

Перейдите в раздел **Звонки и сообщения** → **Мессенджер** → **Настройки мессенджера**.

<img class="miko-shadow img-zoomable"  
src="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_1.png"
data-original="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_1.png"
srcset="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_1.png 1x, /assets/root-guides/connecting_messengers/connecting_max/connecting_max_1.png 2x"
alt="МИКО: Настройки интеграции 1С с MAX, Telegram и WhatsApp"
/>

Нажмите **Добавить** и выберите из выпадающего списка **MAX клиент**.

<img class="miko-shadow img-zoomable"  
src="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_2.png"
data-original="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_2.png"
srcset="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_2.png 1x, /assets/root-guides/connecting_messengers/connecting_max/connecting_max_2.png 2x"
alt="МИКО: Подключение мессенджера для интеграции 1С с MAX"
/>

Откроется форма подключения аккаунта MAX.

Откройте приложение MAX на телефоне и перейдите в раздел Настройки → Устройства → Войти по QR-коду. Затем отсканируйте QR-код, отображаемый в 1С.

<img class="miko-shadow img-zoomable"  
src="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_3.png"
data-original="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_3.png"
srcset="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_3.png 1x, /assets/root-guides/connecting_messengers/connecting_max/connecting_max_3.png 2x"
alt="МИКО:Авторизация QR-кодом в MAX"
/>

Подтвердите создание нового вида контактной информации, если появится соответствующий запрос.

**Разрешите отправлять сообщения первыми**. По умолчанию этот переключатель выключен, то есть диалог с компанией может инициировать только клиент.

<img class="miko-shadow img-zoomable"  
src="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_4.png"
data-original="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_4.png"
srcset="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_4.png 1x, /assets/root-guides/connecting_messengers/connecting_max/connecting_max_4.png 2x"
alt="МИКО: Включение возможности отправлять сообщения первыми из 1С в MAX"
/>

Выберите и укажите **Дежурного сотрудника** в соответствующем поле.

<img class="miko-shadow img-zoomable"  
src="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_5.png"
data-original="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_5.png"
srcset="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_5.png 1x, /assets/root-guides/connecting_messengers/connecting_max/connecting_max_5.png 2x"
alt="МИКО: Выбор дежурного сотрудника для аккаунта MAX в 1С"
/>

Рекомендуем установить **Автоответ на звонок**. Если клиент позвонит через MAX, он получит автоответ с текстом, который вы укажете в данном поле. А сотрудникам в 1С придет уведомление о пропущенном звонке в MAX.

<img class="miko-shadow img-zoomable"  
src="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_6.png"
data-original="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_6.png"

alt="МИКО:  Ввод автоответа на звонок в MAX"
/>

При необходимости укажите **Наименование** для учетной записи.

<img class="miko-shadow img-zoomable"  
src="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_7.png"
data-original="/assets/root-guides/connecting_messengers/connecting_max/connecting_max_7.png"

alt="МИКО:Ввод наименования для подключенного аккаунта MAX в 1С"
/>

Закройте форму. MAX аккаунт успешно подключен к 1С.
