# Подключение MAX бота в 1С

Инструкция по созданию бота в мессенджере MAX доступна по ссылке: 
[!ref text="Создание чат-бота" target="blank"](https://dev.max.ru/docs/chatbots/bots-create)

Перейдите в раздел **Звонки и сообщения** -> **Мессенджер** -> **Настройки мессенджера**.

<img class="miko-shadow img-zoomable"  
src="/assets/root-guides/connecting_messengers/connecting_max_bot/connecting_max_bot_1.png"

alt="МИКО: Настройки интеграции 1С с MAX, Telegram и WhatsApp"
/>

Нажмите **Добавить** и выберите из выпадающего списка **MAX**.

<img class="miko-shadow img-zoomable"  
src="/assets/root-guides/connecting_messengers/connecting_max_bot/connecting_max_bot_2.png"
data-original="/assets/root-guides/connecting_messengers/connecting_max_bot/connecting_max_bot_2.png"
srcset="/assets/root-guides/connecting_messengers/connecting_max_bot/connecting_max_bot_2.png 1x, /assets/root-guides/connecting_messengers/connecting_max_bot/connecting_max_bot_2.png 2x"
alt="МИКО: Подключение бота MAX для интеграции с 1С"
/>

Откроется форма подключения бота MAX.

В поле **Токен** вставьте токен бота MAX и нажмите кнопку **Проверить подключение**. После успешной проверки под кнопкой появится уникальное имя подключенного бота MAX.

<img class="miko-shadow img-zoomable"  
src="/assets/root-guides/connecting_messengers/connecting_max_bot/connecting_max_bot_3.png"
data-original="/assets/root-guides/connecting_messengers/connecting_max_bot/connecting_max_bot_3.png"

alt="МИКО: Ввод токена бота MAX в 1С для подключения"
/>

Подтвердите создание нового вида контактной информации, если появится соответствующий запрос.

При необходимости укажите **Наименование** для учетной записи.

Закройте форму. Бот MAX успешно подключен к 1С.
