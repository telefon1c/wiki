# Настройка модуля маршрутизации на MikoPBX
1. Перейдите в веб-интерфейс АТС, раздел "**Управление модулями**", найдите "**Модуль умной маршрутизации для 1C:Предприятия**" и нажмите кнопку ![](~/assets/rooting/mod_m_1.png).
