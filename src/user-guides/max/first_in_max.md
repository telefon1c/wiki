---
label: Написать клиенту первым в MAX
order: -2
icon: paper-airplane
---

Чтобы подключить возможность первым инициировать диалог с клиентом, перейдите в раздел **Звонки и сообщения → Настройки мессенджера**.

<img class="miko-shadow img-zoomable"

alt="МИКО: Настройки интеграции 1С с MAX, Telegram и WhatsApp"
/> 

В строке с учетной записью мессенджера **MAX** нажмите **Настроить**. В открывшемся окне включите переключатель **Разрешить отправлять сообщение первым**.

<img class="miko-shadow img-zoomable"  
src="/assets/user-guides/max/first_in_max/first_in_max_2.png"
data-original="/assets/user-guides/max/first_in_max/first_in_max_2.png"
srcset="/assets/user-guides/max/first_in_max/first_in_max_2.png 1x, /assets/user-guides/max/first_in_max/first_in_max_2.png 2x"
alt="МИКО: Настройка возможности отправлять сообщения первыми из 1С в MAX"
/>

<img class="miko-shadow img-zoomable"  
src="/assets/user-guides/max/first_in_max/first_in_max_3.png"
data-original="/assets/user-guides/max/first_in_max/first_in_max_3.png"
srcset="/assets/user-guides/max/first_in_max/first_in_max_3.png 1x, /assets/user-guides/max/first_in_max/first_in_max_3.png 2x"
alt="МИКО: Включение переключателя Разрешить отправлять сообщение первым в 1С для аккаунта MAX"
/>

После включения функции откройте карточку клиента. Написать клиенту в MAX можно одним из следующих способов:

* выберите **Сообщение → MAX**;

<img class="miko-shadow img-zoomable"  
src="/assets/user-guides/max/first_in_max/first_in_max_4.png"
data-original="/assets/user-guides/max/first_in_max/first_in_max_4.png"
srcset="/assets/user-guides/max/first_in_max/first_in_max_4.png 1x, /assets/user-guides/max/first_in_max/first_in_max_4.png 2x"
alt="МИКО: Открыть окно для отправки сообщения из 1С в MAX, кликнув на Сообщение → MAX"
/>

* в блоке **Контактная информация** нажмите на номер телефона клиента и выберите **Написать в MAX**;

<img class="miko-shadow img-zoomable"  
src="/assets/user-guides/max/first_in_max/first_in_max_5.png"
data-original="/assets/user-guides/max/first_in_max/first_in_max_5.png"
srcset="/assets/user-guides/max/first_in_max/first_in_max_5.png 1x, /assets/user-guides/max/first_in_max/first_in_max_5.png 2x"
alt="МИКО: Открыть окно для отправки сообщения из 1С в MAX, кликнув на номер телефона → Написать в MAX"
/>

* в блоке **Контактная информация** нажмите на логин пользователя MAX.

<img class="miko-shadow img-zoomable"  
src="/assets/user-guides/max/first_in_max/first_in_max_6.png"
data-original="/assets/user-guides/max/first_in_max/first_in_max_6.png"
srcset="/assets/user-guides/max/first_in_max/first_in_max_6.png 1x, /assets/user-guides/max/first_in_max/first_in_max_6.png 2x"
alt="МИКО: Открыть окно для отправки сообщения из 1С в MAX, кликнув на логин пользователя MAX"
/>

В открывшемся окне **Отправка сообщения** при необходимости выберите из выпадающего списка телефон клиента, зарегистрированный в мессенджере. Затем введите текст сообщения и нажмите **Отправить**.

<img class="miko-shadow img-zoomable"  
src="/assets/user-guides/max/first_in_max/first_in_max_7.png"
data-original="/assets/user-guides/max/first_in_max/first_in_max_7.png"
srcset="/assets/user-guides/max/first_in_max/first_in_max_7.png 1x, /assets/user-guides/max/first_in_max/first_in_max_7.png 2x"
alt="МИКО: Окно Отправка сообщения в 1С для возможности инициировать диалог первыми в MAX, Telegram и WhatsApp"
/>

Также можно использовать шаблоны сообщений. Для этого в форме отправки сообщения нажмите ссылку **Шаблон** и выберите нужный вариант из списка. Поле с текстом сообщения автоматически заполнится текстом выбранного шаблона.
